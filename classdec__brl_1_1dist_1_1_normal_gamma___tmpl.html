Class Representing Normal-Gamma distributions.

Template Parameters
RealTypescalar type used for parameters and return values.
PolicyBoost.Math policy used to calculate results. This effects result accuracy, but the default policy is normally suffice. In Bayesian Analysis, this class of distribution is conjugate for Normal distributions with unknown mean and variance.
