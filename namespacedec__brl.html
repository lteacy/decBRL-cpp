Calculates the Value of Perfect Information (VPI) analytically, given that an actions value distribution is a noncentral t distribution.

In particular, the value distribution takes this form in Bayesian Q-learning. This return value is calculated according to Teacy et al's solution.

Template Parameters
RealTypescalar type used for parameters and return values.
PolicyBoost.Math policy used to calculate results. This effects result accuracy, but the default policy is normally suffice.
Parameters

[in]bestVal1the expected value of the 1st best action.
[in]bestVal2the expected value of the 2nd best action.
[in]valDistthe parameter distribution for the action for which VPI is to be calculated.
Returns

Calculates the Value of Perfect Information (VPI) using monte carlo sampling.

This method is approximate, but works for any value distribution from which random values can be sampled.
